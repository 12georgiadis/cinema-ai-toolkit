# Workflow Overview

A-to-Z pipeline for auteur documentary/experimental filmmaking.

Each phase has its own document. This is the map.

## Phases

```
01-dev          Development: concept, research, writing
02-preproduction    Pre-production: bible, logistics, Obsidian vault
03-shoot        Production: camera, sound, logging
04-ingest       Ingest: media management, backup, proxy generation
05-edit         Editing: assembly, rough cut, fine cut
06-grade        Color grading
07-sound        Sound design and mix
08-transcription    Transcription, subtitles, captions
09-vfx          Visual effects and motion graphics
10-deliver      Export, mastering, distribution
11-archive      Archival and long-term storage
```

## Cross-cutting concerns

- **Claude Code integration** at every phase → see `claude-code/`
- **Obsidian vault** as production bible → see `obsidian/`
- **Notion** for collaborative tracking → see `workflow/notion-bridge.md`
- **Version control** for everything text-based (scripts, FCPXML, metadata)

## Key principle

Every step that can be automated, should be. Every step that requires creative judgment, should be supported by tools that surface the right information at the right time. The filmmaker's time is spent on decisions, not file management.
