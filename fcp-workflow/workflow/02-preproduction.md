# 02 - Pre-production

## Equipment Setup

### Camera
- Canon with **Magic Lantern** firmware (raw video, no recording limit, advanced features)
- Or any camera that shoots in a supported codec (BRAW via BRAW Toolbox, ProRes, H.265)
- **Open source cinema spirit**: understand your tools at firmware level

### Audio
- Separate audio recorder recommended (Zoom, Sound Devices)
- **Sync-N-Link** for multi-camera/audio sync in FCP
- Timecode sync if available, otherwise Sync-N-Link handles waveform matching

### Media
- Fast SD cards / CFexpress for raw recording
- On-set backup to portable SSD (ACASIS M.2 NVMe dock)
- **3-2-1 backup rule**: 3 copies, 2 media types, 1 offsite

## Location scouting

### Documentation
- Photo/video reference → Obsidian vault with **Media Extended**
- GPS coordinates, light conditions, sound environment notes
- Link location notes to relevant scenes in the Fountain script

### Permits and legal
- Track in Notion (shared with production team)
- Claude Code can draft permit applications, releases, contracts from templates

## Crew preparation

### Shared documents
- Notion workspace: call sheets, schedules, contact lists
- Daily call sheets auto-generated from Notion databases
- Equipment checklists per setup

### Communication
- Frame.io for reference sharing (clips, mood boards)
- Notion for async communication
- Signal/Telegram for real-time

## Logging strategy

### Plan your metadata BEFORE shooting
- Define keywords taxonomy in advance
- Create Smart Collection templates in FCP
- Set up **Lumberjack System** for real-time logging during interviews
- Define marker colors: Red = problem, Green = gold, Blue = B-roll, etc.

### Lumberjack System
- **iOS app + web interface**: log during interviews in real time
- Keywords, notes, ratings applied WHILE shooting
- Generates FCPXML with string-outs → import directly into FCP
- $25/month or $110/year
- Invaluable for documentary with hours of interview footage
